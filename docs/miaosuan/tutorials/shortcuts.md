# 对抗界面快捷键

| 按键 | 说明 |
| --- | --- |
| a | 攻击 |
| s | 机动 |
| x | 掩蔽 |
| c | 夺控 |
| f | 间瞄 |
| v | 取消间瞄 |
| q | 切换武器 |
| w | 切换被引导算子 |
| e | 打开通视工具，地对地模式 |
| esc | 取消测量 |
| b | 打开任务 |
| c + 鼠标左键任意六角格 | 下达敌人位预测标记点 |

注意

- 打开通视测量工具后，单击一点开始，双击另一点结束测量。一次测量之后，测量工具仍处于激活状态，可同时进行多次测量。关闭工具请再次点击快捷键或鼠标右键或esc。
