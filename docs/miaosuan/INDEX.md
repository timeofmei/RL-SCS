## 庙算·陆战指挥官是一个战术级实时制陆战兵棋平台，同时是一个智能博弈AI开发平台。

庙算·陆战指挥官支持人类选手进行日常训练和支持大规模在线竞赛的组织与对抗。另一方面，本平台为AI提供开发与评估环境，支持AI通过本平台分发的离线环境实现单机上的开发和调试。同时提供了AI的在线上传测试等功能，以便在网络化平台上开展机机、人机、和人机混合等模式的实时对抗与测评。
